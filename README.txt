1. Chỉnh sửa cài đặt thông qua: 
    Nhấp đúp [ConfigAutoCopyFolder.exe] (Mọi thiết lập sẽ tự động lưu).

2. Khởi động chạy ngầm (chỉ chạy 1 lần duy nhất): 
    Nhấp đúp [src] để mở thư mục src.
    Nhấp đúp [_RUN-AUTO.bat] để khởi động.
